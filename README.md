# Enterprise Knowledge Copilot

An on-prem Retrieval-Augmented Generation (RAG) system for semantic search and question answering over large enterprise documents.

## Features
- PDF ingestion and text preprocessing
- Intelligent chunking and embedding generation
- Vector-based semantic retrieval using ChromaDB
- Grounded LLM responses with source citations
- Model-agnostic, privacy-friendly architecture

## Tech Stack
- Python
- LangChain
- Sentence-Transformers
- ChromaDB
- Open-source LLMs (via Ollama)

## How it works
1. Ingest and clean documents
2. Chunk and embed text
3. Store embeddings in vector database
4. Retrieve relevant context for a query
5. Generate grounded answers using an LLM
